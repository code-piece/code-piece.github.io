Oracle convert json to table

Input:

[
  {
    "id": 1,
    "name": "Lucian",
    "age": 18
  },
  {
    "id": 2,
    "name": "Caitlyn",
    "age": 25
  },
  {
    "id": 3,
    "name": "Soraka",
    "age": 28
  }
]

Output:

IDNAMEAGE
1Lucian18
2Caitlyn25
3Soraka28
SELECT
   id,
   name,
   age
FROM
   JSON_TABLE ( '[{"id":1, "name":"Lucian","age":18},{"id":2,"name":"Caitlyn","age":25},{"id":3,"name":"Soraka", "age":28}]', '$[*]'
       COLUMNS (
           id NUMBER PATH '$.id',
           name VARCHAR2 ( 50 ) PATH '$.name',
           age NUMBER PATH '$.age'
       )
   );
